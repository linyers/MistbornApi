Obten todos los personajes

Para obtener a todos se usa:

/v1.0/characters

Obten solo un personaje

Para obtener a solo uno se necesita solo el id como parametro:

/v1.0/characters/1

Obten varios personajes

Para obtener a varios se necesita el id de los personajes como parametro separados por comas ",":

/v1.0/characters/1,2,3

Filtrar personajes

Tambien se pueden filtrar mediante la URL para incluir mas parametros. Para empezar a filtrar introduce ? seguido por la consulta, por ejemplo: characters/?name=wax. Ademas puede encadenar varias consultas en la misma llamada usando &.

Por ejemplo para saber cuantos kandras son seguidores del supervivencialismo seria: characters/?ethn=kandra&relig=survivorism

Los filtros que se pueden aplicar son:

  • name: Para filtrar por el nombre.

  • ethn: Para filtrar por la etnia.

  • relig: Para filtrar por la religion.
